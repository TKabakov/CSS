display:flex

The seven flex containers will run form letf to right starting in the left corner

justify-content

Aligns the items on main axis.

flex-wrap

The default for this property is "nowrap", which means the item will span the entire width of axis.

When flex-wrap is set to wrap the items will be wrapped to the size of the available viewport.

flex-direction

This property is used to set the main axis, which is "row" by default.
It basically means you are changing your main axis from horizontal rows to vertical columns

align-items

It sets the alignment of the cross axis

align-self

It sets the alignment of individual item inside the flex.

gap:

The gap property can be used to create space between the items along the main axis.

You can also individually configure the gap in rows and columns using row-gap and column-gap

flex

The sub-properties ot flex are flex-grow, flex-shrink and flex-basis.
Together these determine how the flex takes up space, grows or shrinks according the space available.
